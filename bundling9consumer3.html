Single Seller / Bundling / 2 to 9 Consumer Types

Purpose: To compute the single-seller's profit-maximizing strategy regarding bundling (quantity discount). That is, selling several units of the same product as a single package (bundle). The user (you) input willingness to pay for each additional unit of the good by each consumer type, as well as the number of consumers of each type, and the seller's cost parameters.
Then, How-to-PriceTM computes whether bundling is profitable, and how to price bundles as well as each unit when sold unbundled.
