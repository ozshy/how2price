How do I use this site?

If you have training in microeconomics, you shouldn't have any problem understanding what How-to-Price can do for you.  If you don't, our on-line glossary provides some basic definitions of the terms used for the computations.

Many pricing calculations are based on the knowledge of the relevant market demand functions.  If you don't know the market demand functions, you must estimate them using basic regression techniques which are available on most spreadsheet programs.  Regressions of demand functions can also be run on-line in this site.

Why do I get lots of NaN after clicking the "Show Results Now" button?

NaN means "Not A Number" in JavaScript.  You must click the "Reset" button even if you make changes of the default values.

Why the "Results" are not printed with Netscape 4.x?

After you click the button "show results now!", the results are displayed in a different window.  The print button on this "results" window prints well the results using Internet Explorer.  However, for some reason, Netscape 4.x prints the "inputting" window instead of the "results" window.  To solve this problem use IE or Netscape 6.x+ (not yet tested).

It should be mentioned that all windows that do not display results print well on all browsers!  The above comment applies only to the "results" windows.
